# Kurumi

![Kurumi](https://static.zerochan.net/Tokisaki.Kurumi.full.3463699.jpg)

*Kurumi es un proyecto de código abierto enfocado en la gestión de facturas y el control administrativo de pequeños y medianos negocios.*

## Descripción

Kurumi nace con el objetivo de proporcionar una solución sencilla, accesible y gratuita para la administración de facturas, ventas y registros comerciales. El proyecto busca facilitar la organización de la información financiera de un negocio mediante una interfaz intuitiva y herramientas prácticas para el día a día.

Actualmente, el proyecto se encuentra en una etapa temprana de diseño y planificación, por lo que muchas funcionalidades aún están en desarrollo.

## Características planeadas

* Gestión de facturas de compra y venta.
* Registro y consulta de clientes.
* Historial de transacciones.
* Control básico de ventas.
* Exportación de información para respaldo.
* Sistema de informes y estadísticas.
* Generación de reportes mensuales asociados a las ventas del negocio.

## Estado del proyecto

🚧 En fase de diseño.

Las funcionalidades descritas en este documento representan la visión inicial del proyecto y pueden cambiar conforme avance el desarrollo.

## Objetivos

* Crear una herramienta de código abierto para la administración de negocios.
* Mantener una arquitectura sencilla y escalable.
* Facilitar futuras contribuciones de la comunidad.
* Incorporar nuevas funcionalidades de análisis e informes conforme el proyecto madure.
